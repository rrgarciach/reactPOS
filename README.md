# reactPOS: Point Of Sale
## A Native ReactJS mobile application
##### (WORK IN PROGRESS)
This is a simple Point Of Sale for iOS and Android platforms that runs as a stand-alone application.

## Installation
- Install Native React:
`` npm i native-react -g ``
- Clone this repo:
`` git clone https://github.com/rrgarciach/reactPOS ``
- To run in iOS:
`` native-react run-ios ``
- To run in Android:
`` native-react run-android ``
